How To Make A Website

Their are a thousand ways to make a website. In this tutorials I'm going to be teaching the simplest way to make a website because I feel like a lot of the tutorials online overcomplicate the subject at hand. If you want to make a website it's fairly simple to build one. All you have to do is create a new folder and compile.

Visual Studio Code A Really Good Text Editor

In order to start writing code you're going to need a text editor. I suggest installing Visual Studio Code because it provides you with a lot of useful tools, but any of them will do.

Step 1: Create A Folder

A Website is a program and a program is essentially just files in a folder.

The first thing you're going to be doing is open up the folder in your text editor.

Step 2: Creating An HTML File

The main type of file you'll be using when you're building websites are .HTML files.
